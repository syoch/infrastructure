Android 端末のプロビジョニングを快適に

Obtainium を活用して、必要なアプリや自作/野良 APK を一括インストール・自動アップデート管理するためのローカルリポジトリです。

Obtainium 一括インポート用 JSON

ダウンロードした JSON を Obtainium の Import / Export → Obtainium Import で選択すると、全アプリを一発で登録できます。

Obtainium 用 JSON をダウンロード
このサイトは Cloudflare Access で保護されています。アクセス時に表示される認証画面で、利用可能な任意の ID プロバイダー (メール OTP / パスキー / Tailscale SSO 等) でサインインしてください。

新規端末セットアップ手順

01

Obtainium をインストール

まずは更新マネージャーとなる Obtainium を端末にインストールします。下のリンクから最新の APK を直接ダウンロードできます。

GitHub から入手
02

設定ファイルをインポート

Obtainium を開き、メニュー下部の 「Import / Export」 へ進みます。 「Obtainium Import」 を選択し、上でダウンロードした JSON ファイルを選びます。

設定やテーマを含めたい場合は、JSONファイルとしてダウンロードしてインポートすることも可能です。
03

一括インストールと更新設定

インポートが完了すると、アプリ一覧に登録されます。Shizuku (Sui) 等を連携させておくことで、完全にバックグラウンドでのサイレント自動更新が可能になります。
